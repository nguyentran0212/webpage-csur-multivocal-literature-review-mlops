# A Multivocal Review of MLOps Practices, Challenges and Open Issues — Landing Page

Jekyll-based landing page for:

> **Eken, B., Pallewatta, S., Tran, N. K., Tosun, A., & Babar, M. A.** (2025).
> *A Multivocal Review of MLOps Practices, Challenges and Open Issues.*
> ACM Computing Surveys, Vol. 58, Issue 2, Article 1, 1–35.
> DOI: [10.1145/3747346](https://doi.org/10.1145/3747346)

## Local preview

```bash
bundle install
bundle exec jekyll serve --host 127.0.0.1 --port 4001
```

Then visit <http://127.0.0.1:4001/webpage-csur-multivocal-literature-review-mlops/>.

## Deploy

Push to `main`. GitHub Actions builds and deploys to GitHub Pages
automatically (see `.github/workflows/jekyll.yml`).
